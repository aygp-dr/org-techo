* Overview

org-techo brings the philosophy of Japanese paper planners (like Hobonichi Techo) to Emacs org-mode.

** Features

- Daily pages with consistent structure
- Habit tracking and reflection prompts
- Time blocking with grid-like precision
- Minimalist, distraction-free interface
- Exportable to various formats

* Installation

#+begin_src emacs-lisp
;; Add to your init.el
(add-to-list 'load-path "/path/to/org-techo")
(require 'org-techo)

;; Configure
(setq org-techo-directory "~/org/techo")
(setq org-techo-daily-habits
      '("Morning pages" "Exercise" "Reading" "Meditation"))

;; Keybinding
(global-set-key (kbd "C-c t") 'org-techo-goto-today)
#+end_src

** use-package

#+begin_src emacs-lisp
(use-package org-techo
  :load-path "/path/to/org-techo"
  :bind ("C-c t" . org-techo-goto-today)
  :custom
  (org-techo-directory "~/org/techo")
  (org-techo-daily-habits '("Morning pages" "Exercise" "Reading" "Meditation")))
#+end_src

* Usage

| Command                       | Description                        |
|-------------------------------+------------------------------------|
| =M-x org-techo-goto-today=    | Open today's daily page            |
| =M-x org-techo-create-daily-page= | Create a daily page for any date |

* Philosophy

org-techo embraces the minimalist philosophy of Japanese planning:

- *Ichigo ichie* (一期一会) - One day, one page
- *Grid-based structure* for precision
- *Reflection prompts* for mindfulness
- *Habit tracking* for consistency
- *Minimal cognitive overhead*
